智能监控系统

工业自动化监控系统 · 上位机平台

C# / WinForms PLC 通信 SQL Server LiveCharts

项目背景

我围绕制造业自动化产线开发上位机监控平台，用来采集多区域设备状态、报警信息和生产数据，帮助现场人员更快发现异常、定位问题并追溯历史记录。

技术实现

我使用 C# / .NET Framework / WinForms 完成客户端与界面交互，用 PLC 通信库和 NModbus 采集设备数据，用 SQL Server 存储历史数据与报警记录，用 LiveCharts 做趋势与状态可视化，并通过 log4net 归档运行日志。

我的工作

独立交付

完成需求拆解、界面设计、核心功能开发、现场调试和持续优化。

设备采集

负责 PLC 状态读取、设备状态解析、报警规则判断和历史数据记录。

监控看板

实现设备状态、报警弹窗、产量趋势、排班信息和多区域视图切换。

稳定性优化

封装采集任务、UI 刷新、日志归档和窗体生命周期管理，减少卡顿和任务泄漏。

解决的问题

通信

PLC 数据采集与解析

通过 PLC 通信库和 NModbus 采集多区域设备状态，并统一处理不同数据格式。这个部分证明我不仅会做页面，也理解工业设备通信和现场数据结构。

看板

现场可视化

使用 WinForms + LiveCharts 展示设备状态、产量趋势、报警信息和运行指标。现场人员可以通过看板快速判断设备状态，而不是只看原始数据。

稳定

异步任务管理

使用异步任务和 CancellationToken 管理采集生命周期，集中批量刷新 UI。这样能减少界面卡顿，也避免窗口切换后后台任务继续泄漏。

维护

日志与追溯

使用 SQL Server 存储历史数据，结合 log4net 进行日志归档，便于问题排查。系统出现异常时，可以通过报警记录和日志快速还原现场情况。
