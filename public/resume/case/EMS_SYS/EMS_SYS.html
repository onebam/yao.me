智慧能源管理系统

工业园区能源监控平台 · 工业物联网

Blazor Server Modbus SQL Server ECharts

项目背景

我参与开发面向工业园区能源管理场景的 Web 监控平台，用来采集水、电、气等设备数据，实现能源计量、实时看板、异常检测和设备运维辅助。

技术实现

我使用 Blazor Server 构建 Web 监控页面，用 Modbus TCP/RTU 接入工业设备，用 SQL Server + FreeSql 完成数据持久化和查询，用 ECharts 展示能源趋势，并结合 Welford 算法识别能耗波动异常。

我的工作

核心功能

参与设备采集、数据解析、页面展示、异常报警等核心模块开发。

通信接入

对接 Modbus TCP/RTU 设备，完成寄存器映射、数据缩放和状态解析。

Web 看板

使用 Blazor Server 构建电气、水务、燃气等分类监控页面。

配置优化

优化设备参数配置、数据刷新、异常检测和告警推送流程。

解决的问题

通信

Modbus 双协议适配

封装 Modbus TCP/RTU 通信逻辑，适配不同工业设备和网络连接方式。这个能力适合能源管理、设备采集和工业物联网类系统。

数据

数据持久化与看板

使用 SQL Server + FreeSql 管理能源数据，并通过 ECharts 展示实时趋势。系统把原始设备数据转化为可读的能耗指标和管理看板。

可靠

工业级连接机制

支持断线重连、重试机制、连接超时和读取超时控制，降低通信异常影响。工业场景设备不稳定时，系统仍需要尽量保持可用。

告警

能耗异常检测

结合 Welford 在线算法与阈值策略识别能耗波动异常，辅助运维判断。这让系统不只是展示数据，也能对异常用能进行主动提示。
